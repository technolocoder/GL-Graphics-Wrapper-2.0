Dependencies:
    SDL2
    OpenGL GLEW
    GLM OpenGL Math Library
    STB Image Loader
    Assimp Model Loader 
-----------------------------

Setup: 
    Use "make install_dependencies" command

Run Demo: 
    Use "make run APP=app_name" where you replace "app_name" with the available samples.

Samples:
    test-rect
    texture-load

clean:
    Use "make clean"
